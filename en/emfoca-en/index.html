EMFOCA - E-commerce website

I redesigned an E-commerce website for EMFOCA, a coaching business.

2021

Freelance

Check the live site here.

The client

EMFOCA is a recently founded life and business coaching company in Lima. Their digital marketing strategy included updating their marketing site to an E-commerce website that would serve as a platform to sell their courses and sessions.

The Project

The main goal of this project was to update the website and replace their old WordPress marketing website. The new site needed to be more usable, cleaner and capable of hosting or selling digital products.

Some key considerations for this project:

  • The new site had to be more modern, less bloated and overall cleaner.
  • The store side of the website had to be accessible from the main menu.
  • The site had to be fully responsive.
  • Images had to be free to use.
  • The final website had to be easy to edit and modify for future updates.

In addition to that, the brand itself needed a revamp to fit the new specifications of the website.

The Process

My first task was to update the old logo. The client wanted to maintain the core concept with updated colours and typography.

I redid the isotype, giving it more movement to fit the newer, more modern identity they wanted to take. The font was also updated.

I organized all the website information and functions into 6 categories:

During the first meeting, the client expressed interest in having a call to action button redirecting to either the contact page or shop page on all the other pages.


The client also wanted that all user paths lead to either the contact page or the shop page. To ensure this was the case, I made a bit of an unorthodox user flow mapping out all the possible contact points on the site.

With all that information, and once the user flow was completely approved, I proceeded to make the very first low-fi wireframes of the site.

After the wireframes were approved, I started working on the visual side of the site. To develop the new design system, I used the new brand colours as a starting point, using variant shades as the primary and secondary UI colours and selecting compatible colours for any semantic meanings.

EMFOCA is a coaching business, so it was indispensable to showcase positive imagery and photography throughout the site. This positivity was also reflected in the colour palette, having light and vibrant colours as the main and accents. The new visual style has a fresher look&feel as opposed more "corporate" colour scheme of the old site.

I include a screenshot of the old website for reference:

I chose stock photos that reflected the brand's values and updated visual image. Prioritizing images of people, hands and collaborative actions over corporate setting images.

For client revision and feedback, I made a quick one-page HTML prototype site to showcase the visual style live. The site is still available here (thru Netlify).

With all the considerations settled, I designed the final screens for mobile and desktop and then started building the site.

The approved style guide:

The old site was a WordPress site, and the client preferred to continue using that service. For convenience sake, I chose a bootstrap based framework with a built-in SASS compiler to build a custom theme for the site. (bootScore with wooCommerce integration).

It was important for the client to keep the site fairly simple to edit from the WordPress interface and Gutenberg blocks editor, so I kept all the pages that needed code edits at a minimum.

Still, I had to edit some things in PHP. Specifically, a new layout for the shop page and a function removing the zoom-in-product-photo feature (which is unnecessary in a website focusing on selling digital products and courses)

After testing the site with six users and fixing some bugs with the payment gateway, the site went live in September 2021. You can check out the live site here.
