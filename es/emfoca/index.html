EMFOCA - Web Ecommerce

Rediseñé un sitio web E-commerce para EMFOCA, una empresa de coaching.

2021

Freelance

Puedes ver la web final aquí.

El cliente


EMFOCA es una empresa de coaching de vida y negocios recientemente fundada en Lima. Su estrategia de marketing digital incluía la actualización de su sitio web e-commerce para que sirviera como una plataforma para vender cursos y sesiones.

El proyecto

El objetivo principal de este proyecto era actualizar el sitio web para reemplazar el antiguo sitio de marketing hecho en WordPress. El nuevo sitio tenía que ser más user-friendly, limpio y capaz de alojar o vender productos digitales.

Algunas consideraciones clave para este proyecto:

  • El nuevo sitio tenía que ser más moderno, menos sobrecargado y más limpio en general.
  • La tienda del sitio web tenía que ser accesible desde el menú principal.
  • El sitio tenía que ser completamente responsive.
  • Las imágenes tenían que ser de uso gratuito.
  • El sitio web final tendría que ser fácil de editar y modificar para futuras actualizaciones.

Además de todo esto, la marca en sí necesitaba una renovación para adaptarse a las nuevas especificaciones del sitio web.

Proceso

Mi primera tarea fue actualizar el logotipo antiguo. El cliente quería mantener el concepto, solo con colores y tipografías actualizadas. Rehice el isotipo, dándole más movimiento para que se adapte a la identidad más nueva y moderna que querían tomar. La fuente también se actualizó.

Durante la primera reunión, el cliente expresó interés en tener un botón CTA en todas las páginas que redirigiera a la página de contacto o la página de la tienda.

El cliente también quería que todas las posibles rutas del usuario terminaran o en la página de contacto o  la de la tienda. Para asegurarme de que este fuera el caso, hice un flujo de usuario (un poco experimental) mapeando todos los posibles puntos de contacto en el sitio.

Con toda esta información, y una vez que el flujo de usuario fue completamente aprobado, procedí a hacer los primeros wireframes de baja fidelidad del sitio.

Después de que se aprobaron los wireframes, comencé a trabajar en el lado visual del sitio. Para desarrollar design system, utilicé los nuevos colores de la marca como punto de partida. La paleta final incluyó colores principales derivados de los colores de la marca y colores complementarios para cualquier significado semántico.

EMFOCA es un negocio de coaching, por lo que era indispensable mostrar imágenes y fotografías positivas en todo el sitio. Esta positividad también se debía reflejar en la paleta de colores, teniendo colores claros y vibrantes como protagonistas y acentos. El nuevo estilo visual tuvo un aspecto y una sensación más fresca en comparación con el estilo de color más "corporativo" del sitio anterior.

Elegí fotografías de archivo que reflejaran los valores de la marca y la identidad visual actualizada. Prioricé imágenes de personas enteras o manos y acciones colaborativas por sobre imágenes meramente corporativas.

Hice un prototipo  rápido HTML de una sola página para que el cliente pudiera ver el estilo visual en vivo. El sitio todavía está disponible aquí (a través de Netlify).

Con todas las consideraciones resueltas, diseñé las pantallas finales para varios tamaños y comencé a construir el sitio.

La guía de estilo aprobada:

El sitio anterior estaba hecho en WordPress y el cliente prefirió que se esto mantuviera así. Por conveniencia, elegí un framework basado en bootstrap con un compilador de Sass incluído para crear un tema personalizado del sitio. (bootScore con integración de wooCommerce).

Era importante para el cliente que el sitio fuera fácil de editar desde la interfaz de WordPress y el editor de bloques de Gutenberg, así que mantuve todas las páginas que pude con el menor código posible. Aún así, tuve que editar algunas cosas en PHP. Específicamente, alterar el layout para la página de la tienda y agregar una función que eliminara la acción de zoom-in en las fotos de producto (que es innecesaria en un sitio web que se centra en la venta de productos y cursos digitales). Después de probar el sitio con seis usuarios y corregir algunos errores con la pasarela de pago, el sitio se puso en marcha en septiembre de 2021. Puede entrar el sitio activo aquí.
